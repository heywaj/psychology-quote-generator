# 字体下载说明

## SmileySans-Oblique 字体

这个项目使用 SmileySans-Oblique 字体来获得最佳显示效果。

### 下载方式：

1. **GitHub 官方仓库**：
   - 访问：https://github.com/atelier-anchor/smiley-sans
   - 下载 SmileySans-Oblique.ttf 文件

2. **直接下载链接**：
   - https://github.com/atelier-anchor/smiley-sans/releases

### 安装步骤：

1. 下载 `SmileySans-Oblique.ttf` 文件
2. 将文件放入 `resources/fonts/` 目录下
3. 重新运行程序即可

### 备注：

- 如果没有安装此字体，程序会自动使用系统默认字体（微软雅黑）
- 虽然可以正常运行，但显示效果可能不如专用字体
- SmileySans 是一个开源的中文字体，支持简体中文、繁体中文等
